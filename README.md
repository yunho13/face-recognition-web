# Face Recognition Web Dashboard

This project implements a face recognition system using Python, Hugging Face models, and a web dashboard.

## Features
- Real-time camera integration
- Face detection and recognition using ML models
- Web-based dashboard for monitoring

## Setup
1. Create a virtual environment: `python -m venv venv`
2. Activate the environment: `source venv/bin/activate` or `venv\Scripts\activate`
3. Install dependencies: `pip install -r requirements.txt`
